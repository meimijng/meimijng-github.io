windows下配置git中的vim

Gvim下修改配置

安装好Gvim后，可以找到安装目录下的 _vimrc 文件（例如：我的在 C:\Program Files (x86)\Vim\）
添加一些自己常用的配置，即可在Gvim中生效

Git中的vim编辑器的配置修改

一直使用git官网的Git for Windows，习惯用bash，使用自带的Vim编辑器总是想改配置，但是一直不知道如何修改，今天研究了一下如何去修改配置，首先到Git的安装目录（我的是 C:\Program Files\Git\ ）,在目录 C:\Program Files\Git\usr\share\vim\vim74\ 找到了vim的安装文件，然而并没有发现配置文件，没办法只能在 git 的安装目录下搜索 vimrc 之后定位到了 C:\Program Files\Git\etc\ 目录，测试了一下，添加了显示行号的配置，在git bash中测试使用vim编辑器，确实显示出了行号。
总结：只用修改git安装目录下的 etc 目录中的 vimrc 即可达到修改 git 中 vim 编辑器配置的效果.
